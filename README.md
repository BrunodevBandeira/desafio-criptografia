## Tecnologias 

- Spring Boot
- Spring MVC
- Spring JPA
- H2
- Spring security

 ## Sobre o desafio

 Implementação de criptografia transparente para campos sensíveis em entidades, onde os campos são entregues descriptografados nas requisições das APIs e persistidos criptografados no banco de dados.
